# Keyboard Shortcuts & Mouse Controls

## Global

| Shortcut | Action |
|---|---|
| **Tab** | Switch between CAM View and Control View |
| **Ctrl+Q** | Quit the application |
| **Escape** | Abort current operation (Control View) |

## Control View -- Jogging

| Shortcut | Action |
|---|---|
| **Arrow keys** | Continuous jog (hold to move, release to stop) |
| **Ctrl+Arrow keys** | Precise jog (moves an exact distance set in [Machine Parameters](machine-parameters.md)) |
| **Page Up / Page Down** | Z-axis jog |

## Control View -- Mouse

| Input | Action |
|---|---|
| **Left-click** on machine plane | Navigate to waypoint (confirmation dialog) |
| **Ctrl+left-click** on G-code contour | Jump-in: start program from that contour |
| **Ctrl+right-click** on G-code contour | Reverse the contour's cutting direction |
| **Right-click drag** | Pan the view |
| **Scroll wheel** | Zoom in/out |

## CAM View -- Mouse

| Input | Action |
|---|---|
| **Left-click** | Select or edit (depends on active tool) |
| **Right-click drag** | Pan the view |
| **Scroll wheel** | Zoom in/out |
| **Ctrl+scroll** on selected part | Rotate in 5-degree increments |
| **Shift+scroll** on selected part | Scale |
