# NanoCut Control Manual

Welcome to the NanoCut Control manual!

## Overview

NanoCut Control is a CNC control application for plasma cutting machines running
NanoCut firmware.

## Features

- Reliable control based on the grbl protocol
- Real-time machine control
- DXF file import and processing
- G-code generation supporting Torch Height Control

## Quick Start

1. Build the application using CMake
2. Connect to your CNC machine via serial port
3. Load a DXF file or create a new design
4. Generate G-code and start cutting!
